# Johnny Faris - Portfolio (Next.js Migration)

This is a Next.js version of the [Dimension](https://html5up.net/dimension) HTML5 UP template.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Structure

- `src/components`: React components (Header, Main, Footer, Sections).
- `src/styles`: Ported SCSS files.
- `public/images`: Static assets.

## Deployment

This project is configured for static export to GitHub Pages.

```bash
npm run build
# The 'out' folder contains the static site.
```
